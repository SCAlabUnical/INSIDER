API Documentation

Endpoint: POST process_post_data

This endpoint receives data from a POST request, processes it, and returns a JSON response.

Request

Method: POST

Content-Type: application/json

Request Body

{
    "workflow": {...},
    "catalog": {...}
}

Response

Status Code: 200 OK

Response Body

{
    "message": "Workflow and catalog successfully received",
    "result": {...}
}

Code Explanation

The process_post_data function handles the POST request:

  1. Receives the data from the request body and parses it from JSON.
  2. Extracts the workflow and catalog objects from the parsed data.
  3. Calls the compare_workflow_and_catalog function with the extracted data.
  4. Returns a JSON response containing a success message and the result of the comparison.
